## 🍽️ Restaurant Insights from EazyDiner | Web Scraping & Exploratory Data Analysis

This project presents a comprehensive analysis of restaurant data collected from EazyDiner using web scraping techniques and Exploratory Data Analysis (EDA). The objective is to transform raw, unstructured data into meaningful insights that highlight key trends in the restaurant industry.

### 🎯 Objectives
* Extract restaurant data using web scraping methods.
* Perform data cleaning and preprocessing to ensure data quality.
* Conduct Exploratory Data Analysis to identify patterns and relationships.
* Analyze key variables such as ratings, pricing, discounts, and locations.
* Present insights through effective data visualization.

### 🛠️ Tech Stack
* Programming Language: Python
* Libraries: Pandas, NumPy, Matplotlib, Seaborn
* Web Scraping: Requests, BeautifulSoup

### 📌 Key Insights
* Majority of restaurants maintain high ratings between 4.0 and 4.5, indicating strong customer satisfaction.
* Multicuisine restaurants dominate the market, followed by Indian and Pan Asian cuisines.
* Bengaluru and Mumbai exhibit higher average dining costs compared to other cities.
* A strong positive correlation exists between price and final bill amount.
* Discounts have a moderate impact, whereas ratings show minimal correlation with pricing.

### 📈 Conclusion
The analysis demonstrates how data-driven approaches can uncover valuable business insights. By leveraging EDA techniques, this project provides a clear understanding of restaurant trends, pricing behavior, and customer preferences, enabling informed decision-making.
